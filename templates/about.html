About This Dashboard

This dashboard provides a fast, open, and searchable interface for exploring PDF extracts of United States Supreme Court cases. You can search for any word or phrase, and the dashboard will instantly show you all matching opinions with context snippets. Click on any result to view the full PDF text, and use the dark mode toggle for comfortable reading at any time of day.

Built with open data, Python, Flask, and Bootstrap. Source code available on GitHub.
